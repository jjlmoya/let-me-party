# vue3-boilerplate-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### install any of the desired native platforms:
```
npx cap add android
```

```
npx cap add ios
```

### For production don't forgive
```
npm run build
```
for generate files to capacitor
